vuex 学习笔记

发表于2018-03-07|更新于2024-12-17|前端开发Vue.js
|总字数:406|阅读时长:1分钟

  • state

    记录所有数据和状态的，跨页面使用的

  • getters

    store 里的计算属性

  • mutation

    用于修改 state 状态

    mapMutations 辅助函数将组件的方法映射为 store.commit 调用

  • action

    提交 mutation 间接变更状态

    保护一些常用业务操作，通过提交 mutation 改变业务中的状态

    mutation 中则只包含更改 state 的方法

    可以有异步操作

    mapActions  辅助函数将组件的 methods 映射为   store.dispatch  调用

    action 可以返回一个 promise

  • modules

    把 store 分割为模块
