# cesium 的几个小例子

[houxiaozhao/cesium-demo](https://github.com/houxiaozhao/cesium-demo)

### **安装**

使用前先安装 cesium

```bash
npm install cesium
```

其他依赖包括（非必要，为了方便和好看）不需要安装，已经包含在仓库内

- vue
- echarts
- element

### **使用**

anywhere 或其他静态文件服务器

```bash
anywhere
```
